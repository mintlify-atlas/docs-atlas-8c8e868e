## Directory Layout

```python
        self.dhcp_pools       = []
        self.vlans_data       = []
        self.static_routes    = []
        self.ospf_networks    = []
        self.qos_classes      = []
        self.pol_entries      = []
        self.service_policies = []
```

```python
def expand_ports(ports_str: str) -> str:
    """
    Normaliza nombres de interfaces a la forma completa que espera IOS 15.x.

      - "Gi0/1-3"     → "GigabitEthernet0/1-3"
```

```python
# source/core/connector.py:50-69
def detect_l3(nc) -> bool:
    """
    Detecta si el switch tiene ip routing activo (Layer 3) o no (Layer 2).

    Estrategia: ejecuta 'show ip route connected' y busca líneas que comiencen
    con 'C' (ruta directamente conectada). Si las hay, el switch enruta.
    """
    try:
        out = nc.send_command("show ip route connected", delay_factor=2)
        return bool(re.search(r'^C\s+', out, re.MULTILINE))
    except Exception:
        return False   # Si falla, asumimos L2 por seguridad
```
</Accordion>

## ui/ Directory — Presentation Layer

### widgets.py — Styled Widget Factory

**Responsibilities:**
- Define global theme colors and fonts
- Create styled Tkinter widgets (buttons, entries, labels, frames)
- Apply consistent visual design across all tabs
- Provide `apply_style()` function for root window

### tabs_sw_dhcp.py — Tabs ① and ②

**Responsibilities:**
- **Tab ①:** Switch inventory management (add, edit, delete, test connection)
- **Tab ②:** DHCP pool configuration (network, mask, gateway)
- Functions: `build_tab_sw()`, `build_tab_dhcp()`

### tab_vlan.py — Tab ③

**Responsibilities:**
- VLAN creation and naming
- Port assignment (access/trunk mode)
- Associate DHCP pools with VLANs
- Configure VLAN security (block social media, custom ACLs)
- Function: `build_tab_vlan()`

### tab_routing.py — Tab ④

**Responsibilities:**
- Static route configuration (destination, mask, next-hop)
- OSPF configuration (PID, network statements, area)
- Enable/disable `ip routing`
- Function: `build_tab_routing()`

### tab_qos.py — Tab ⑤

**Responsibilities:**
- Class-map configuration (match DSCP, IP, ACL)
- Policy-map entries (priority, bandwidth, police, shape, set dscp)
- Service-policy assignment to interfaces
- Function: `build_tab_qos()`

### tab_exec_backup.py — Tabs ⑥ and ⑦

**Responsibilities:**
- **Tab ⑥:** Deploy configuration to selected switches with real-time logging
- **Tab ⑦:** Export/import JSON, download running-config, upload to AWS S3
- Functions: `build_tab_exec()`, `build_tab_backup()`

### tab_security_gre.py — Tabs ⑧ and ⑨

**Responsibilities:**
- **Tab ⑧:** Enable secret (MD5), login blocking, banner MOTD
- **Tab ⑨:** GRE over IPsec tunnel configuration (IKEv1, ESP-AES-256)
- Functions: `build_tab_security()`, `build_tab_gre()`
- Functions: `build_security_commands()`, `build_gre_ipsec_commands()`

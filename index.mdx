## Welcome to SECURE-BRANCH-lac

SECURE-BRANCH-lac is a comprehensive GUI tool for mass-configuring Cisco IOSvL2 switches across multiple branch offices. Built with Python and Tkinter, it automates network infrastructure deployment with support for VLANs, DHCP, QoS, routing, security hardening, and GRE over IPsec tunnels.

**Target Firmware:** `vios_l2-ADVENTERPRISEK9-M`, IOS Experimental 15.2(20200924:215240)

## Configuration Workflow

SECURE-BRANCH-lac provides a nine-tab interface covering the complete configuration lifecycle:

1. **Sucursales** — Add switches with SSH credentials and test connections
2. **DHCP** — Define DHCP pools (network, mask, gateway)
3. **VLANs y Puertos** — Create VLANs, assign ports (access/trunk), link DHCP pools
4. **Enrutamiento** — Configure static routes, default gateway, and OSPF
5. **QoS MQC** — Define class-maps, policy-maps with police/shape/priority/set DSCP
6. **Ejecución** — Apply configuration to selected switches with real-time logging
7. **Backup** — Export/import JSON, download running-config, upload to AWS S3
8. **Seguridad** — Enable secret, login block-for, banner MOTD
9. **GRE/IPsec** — Configure GRE tunnels encrypted with IPsec

<Note>
All configurations are applied safely with automatic detection of Layer 2 vs Layer 3 switches and built-in protections for uplink interfaces (GigabitEthernet0/0).
</Note>

## Use Cases

- **Multi-site branch office deployment** — Configure dozens of switches consistently
- **Network segmentation** — Isolate departments with VLANs and ACLs
- **QoS implementation** — Prioritize voice, video, and critical data traffic
- **Secure tunneling** — Connect branch offices with GRE over IPsec
- **Disaster recovery** — Automated backups to AWS S3 with encryption
- **GNS3 simulation** — Test configurations in virtual environments before production

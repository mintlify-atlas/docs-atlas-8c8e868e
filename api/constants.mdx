## Overview

The `constants` module centralizes all application-wide constants including version metadata, UI theme colors, protected network interfaces, and QoS DSCP presets. Modify this file to change the visual theme or default configuration values.

**Module Path:** `constants.py`

## Protected Network Resources

### UPLINK_IFACE

Management/uplink interface that must **NEVER** be modified by the application.

```python
UPLINK_IFACE = "GigabitEthernet0/0"
```

<ResponseField name="UPLINK_IFACE" type="str">
  Protected interface: `"GigabitEthernet0/0"`
</ResponseField>

#### Why This Matters

This interface connects the switch to the router/core network. Modifying it could:
- Break SSH access to the switch
- Disconnect the switch from the network
- Require console access to recover

#### Protection Mechanisms

The cleanup function `build_cleanup_commands()` checks every interface against `UPLINK_IFACE` and skips:
- Port mode changes (access/trunk)
- VLAN assignments
- Service-policy associations
- Any configuration modifications

#### Example

```python
from constants import UPLINK_IFACE

def is_protected_interface(iface: str) -> bool:
    """Check if interface should never be modified."""
    return UPLINK_IFACE.lower() in iface.lower()

# Usage in cleanup logic
if is_protected_interface("GigabitEthernet0/0"):
    print("Skipping uplink — protected interface")
    # Skip this interface during cleanup
```

## QoS Configuration

### DSCP_PRESETS

Predefined DSCP (Differentiated Services Code Point) values for QoS quick buttons.

```python
DSCP_PRESETS = [
    ("EF  — Voz/Video (46)",          "ef"),
    ("AF41 — Video conferencia (34)", "af41"),
    ("AF31 — Llamadas (26)",          "af31"),
    ("AF21 — Datos críticos (18)",    "af21"),
    ("AF11 — Datos normales (10)",    "af11"),
    ("CS3  — Señalización (24)",      "cs3"),
    ("CS1  — Scavenger (8)",          "cs1"),
    ("Default — Best effort (0)",     "default"),
]
```

<ResponseField name="DSCP_PRESETS" type="list[tuple[str, str]]">
  List of tuples: `(display_label, ios_value)`
</ResponseField>

#### DSCP Value Reference

| Label | IOS Value | Decimal | Binary | Use Case |
|-------|-----------|---------|--------|----------|
| **EF** | `ef` | 46 | `101110` | Voice/video (low latency, low jitter) |
| **AF41** | `af41` | 34 | `100010` | Video conferencing |
| **AF31** | `af31` | 26 | `011010` | Voice calls |
| **AF21** | `af21` | 18 | `010010` | Critical data (database, ERP) |
| **AF11** | `af11` | 10 | `001010` | Normal business data |
| **CS3** | `cs3` | 24 | `011000` | Call signaling (SIP, H.323) |
| **CS1** | `cs1` | 8 | `001000` | Scavenger (bulk downloads, P2P) |
| **Default** | `default` | 0 | `000000` | Best effort (no priority) |

#### DSCP Naming Conventions

- **EF** = Expedited Forwarding → Highest priority, guaranteed low latency
- **AFxy** = Assured Forwarding → Class `x` (1-4), Drop precedence `y` (1-3)
- **CSx** = Class Selector → Backward compatible with IP Precedence
- **Default** = Best Effort → No QoS guarantees

#### Example

```python
from constants import DSCP_PRESETS

# Display preset options in GUI
for label, value in DSCP_PRESETS:
    print(f"{label} → IOS command: match dscp {value}")

# Output:
# EF  — Voz/Video (46) → IOS command: match dscp ef
# AF41 — Video conferencia (34) → IOS command: match dscp af41
# ...

# Get IOS value for voice traffic
voice_dscp = next(val for lbl, val in DSCP_PRESETS if "Voz" in lbl)
print(f"Voice DSCP: {voice_dscp}")  # Output: Voice DSCP: ef
```

### Checking Protected Interface

```python
from constants import UPLINK_IFACE
import re

def parse_interfaces(config: str) -> list:
    """Extract all interfaces except uplink."""
    interfaces = []
    for iface in re.findall(r'^interface (\S+)', config, re.MULTILINE):
        if UPLINK_IFACE.lower() not in iface.lower():
            interfaces.append(iface)
    return interfaces

# Example
config = """
interface GigabitEthernet0/0
 ip address 192.168.1.10 255.255.255.0
interface GigabitEthernet0/1
 switchport mode access
interface GigabitEthernet0/2
 switchport mode trunk
"""

ifaces = parse_interfaces(config)
print(ifaces)  # ['GigabitEthernet0/1', 'GigabitEthernet0/2']
# GigabitEthernet0/0 (uplink) is excluded
```

### Building QoS Class-Map from Preset

```python
from constants import DSCP_PRESETS

def create_voice_qos_commands() -> list:
    """Generate QoS commands for voice traffic."""
    # Find EF (voice) preset
    ef_value = next(val for lbl, val in DSCP_PRESETS if "Voz" in lbl)
    
    return [
        "class-map match-all VOICE",
        f"match dscp {ef_value}",
        "exit",
        "policy-map QOS-POLICY",
        "class VOICE",
        "priority percent 30",
        "exit",
        "exit",
    ]

cmds = create_voice_qos_commands()
for cmd in cmds:
    print(cmd)

# Output:
# class-map match-all VOICE
# match dscp ef
# exit
# policy-map QOS-POLICY
# class VOICE
# priority percent 30
# exit
# exit
```
